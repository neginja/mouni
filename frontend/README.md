# Mouni - Frontend

Mouni is a lightweight group expenses manager. This is the frontend package written in `dart` with Flutter.

## Install dependencies

To install the dependencies for Mouni, run

```console
flutter pub get
```

To upgrade dependencies

```console
flutter pub upgrade --major-versions
```

## Debug

To debug the frontend the easiest is to directly use flutter tooling with change detection, from `frontend/`:

```console
flutter run
```
